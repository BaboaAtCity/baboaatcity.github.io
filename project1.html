Description

This is an inventory management system for sneaker resellers with its very own REST API for fetching live payout prices from Laced.co.uk . The web application was created using Laravel which is a modern PHP framework and the API was built using Express, NodeJS and axios. I designed the API so that it is able to parse the react prop from the customer side of Laced to then access all the price data and then deduct the amount of fees from each price and then return the payout prices. The inventory system has the following features below :

  • Add/Edit/Delete Item
  • Live update search for an item
  • Generate a link so your inventory can be shared
  • Mark an item as sold with details of sale
  • Live market data from UK market's eBay and Laced
  • Export all sales to CSV
  • Sales analytics on home page
